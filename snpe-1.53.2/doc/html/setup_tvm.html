TVM Setup

A TVM installation is required if the SNPE SDK is going to be used to convert TFlite models.

Get TVM

  • The SNPE SDK has been tested with TVM v0.7
  • TVM v0.7 depends on Python 3.6
  • Instructions for TVM setup are at https://tvm.apache.org/docs/install/from_source.html. Follow those instructions to install TFLite on Ubuntu Linux or see the code below for a quick reference. 
        git clone --recursive https://github.com/apache/incubator-tvm -b v0.7 tvm

    cd tvm
    mkdir build
    cp cmake/config.cmake build

    cd build
    cmake ..
    make -j4
  • Setup the environment using the following commands 
        export PYTHONPATH=<tvm source dir>/python:$PYTHONPATH
    export LD_LIBRARY_PATH=<tvm build dir>:$LD_LIBRARY_PATH

  • Check TVM dependencies at https://tvm.apache.org/docs/install/from_source.html#python-dependencies

    Necessary dependencies: 

        pip3 install --user numpy decorator attrs
  • Environment test: 
        python -c "import tvm"
  • Continue with the rest of the SNPE Setup
